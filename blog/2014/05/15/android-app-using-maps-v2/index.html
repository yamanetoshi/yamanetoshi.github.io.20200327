昨年もごもごした Android のコンテンツアプリ

Git なリポジトリから clone して実行など、と思ったところ Maps V2 使ってるのと開発機の HDD が壊れてリセットされてた事などから偉く面倒な事になっていた。というか作り方くらい控えとけ < 自分

手順を控える、というかリポジトリに反映させるファイルが足りてないのかどうか。とりあえず以下の手順を辿りました。

  • ソースを git clone して Android プロジェクトとして import
  • actionbarsherlock を git clone して import して Library に追加
  • google play service の導入
  • google play service のライブラリプロジェクトを import して Library に追加
  • AndroidManifest に gms.version な meta-data が無い
  • API キーが違う (HDD リセットされてて debug.keystore も変わってる) ので作り直して反映

微妙すぎる。一応動く状態にはなりましたがこれは README つくっとく必要があるレベルですねorz
