Rails 開発環境構築メモ

自分メモです。Dockerfile 転記用、ってことで。

ここも見つつ、ってことで以下。

$ sudo apt-get install -fy build-essential libssl-dev curl git-core libyaml-dev
$ curl -L https://get.rvm.io | bash -s
$ . ~/.profile

で、以下も念の為。

$ sudo apt-get install build-essential openssl libreadline6 libreadline6-dev curl \
git-core zlib1g zlib1g-dev libssl-dev libyaml-dev libsqlite3-dev sqlite3 libxml2-dev \
libxslt-dev autoconf libc6-dev ncurses-dev automake libtool bison subversion

で、2.0 な Ruby を導入。

$ rvm install 2.0.0 --with-openssl-dir=$HOME/.rvm/usr

で、以下か。

$ cat >> .bashrc
source $HOME/.rvm/scripts/rvm
$ . .bashrc

続いて以下。

$ rvm use 2.0.0 --default
$ gem update --system 2.0.3
$ cat > .gemrc
install: --no-rdoc --no-ri
update: --no-rdoc --no-ri
$ gem install rails --version 4.0.4

これで初期設定は完了ってことで良いのかどうか。
