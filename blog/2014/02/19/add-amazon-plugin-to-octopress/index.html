Octopress にあまぞんなソレを設定するなど

以下を確認しつつ設定盛り込みの方向。

  • Amazon Plugin を追加する

とりあえず以下を Gemfile に追加して

gem 'ruby-aaws'

bundle install した後に plugins 方面にファイルを投入とのこと。

$ curl https://raw.github.com/jamuraa/base0.net/master/_plugins/amazon_liquid_tags.rb > plugins/amazon_liquid_tags.rb

で、$HOME に設定ファイルを追加とのこと。ACCESS-KEY とか忘却の彼方ですね。つーかこれって aws なソレなのかな。だったら放置、ということで最近入れたエントリにリンクを突っ込んでみます。

Liquid Exception とか出力されてますね。最後に Build Failed と出ている。
