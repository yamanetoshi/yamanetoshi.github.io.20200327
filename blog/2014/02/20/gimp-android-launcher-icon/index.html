Gimp で android の Launcher icon を作ってみる

とりあえず 512x512 px で新規作成。Image -> Configure Grid にてとりあえず intersections (dots) を選択。View から Show Grid および Snap to Grid にチェック。

なんとなくでっちあげた後、

  • 背景透過に
  • 枠もつけてみた

ということができるようになったみたいなので以下に控え。

背景透過

画像部分を Fuzzy Select Tool で選択して以下の順。

  • アルファチャンネル追加 (今回はスデに追加済みでした)
  • Edit -> Clear

で、余計な部分を全て clear したら保存すれば OK でした。

枠のつけかた

枠を付けたい図形は一つのレイヤにしておいた方が良いみたい。とりあえず枠を付けたい画像のレイヤを右クリックして Alpha to Selection を選択。

  • レイヤを追加して枠を付ける図形なレイヤの下に配置
  • Select -> Grow を選択
  • 今回は 1 px にして OK
  • 範囲を塗り潰し

という操作で枠は作れたみたいなんですが、どうも微妙。
