UI 勉強会をしてみた

在那覇な co-working space であるところの_のまどん_さんにて勉強会やってみませんか? と提案してみたところ、ノッてきてくれたので開催してみました。異業種な方面からのご参加があり、非常に勉強になりました。

今後も何かの形で続けていければ良いな、と思っていますがどうなるか。

今回はスマホ UI って事だったんですが、Web サービス使う、って観点で workshop てきなソレを、ってのはどうかな、って思ってる次第です。

追記

一回り上の方とご一緒させて頂きました。昔一緒に仕事してたセンパイが一回り上だったのですが彼は今どんなになっているのだろう。お元気にしてらっしゃるのかどうか。

あと 5 年くらい経つと体が色々壊れてきますよ、って微妙な助言を頂戴しました。今より微妙になるってどうなの。てか真面目に体のメンテしないと駄目だな。

そしてアフォーダンスという視点というのは自分てきにも非常に気づきのあるもので非常に参考になりました。次回の開催がとても楽しみですがどうなるか。
