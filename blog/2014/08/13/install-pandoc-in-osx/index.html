pandoc を osx に導入

備忘録です。以下から download できる模様。

  • jgm/pandoc

終了したら zip を展開して pandoc-1.12.4.2.pkg を起動したらインストーラが起動しました。処理が終わったら端末起動して以下。

$ which pandoc
/usr/local/bin/pandoc

一応なんとかなってる模様。おもむろに以下なカンジで

$ pandoc -f markdown -t html -o hoge.html hoge.md

問題なく変換できておりました。素晴しす。あとこれって例の雛型に盛り込めんかな。

追記

変換したナニは文字化けていた。何故かな、って思ったら html ヘッダが無かった。色々確認してみるに以下なリポジトリを発見。

  • jgm/pandoc-templates

ここから default.html を取得して以下で変換してみました。

$ pandoc -f markdown -t html -o hoge.html --template default.html hoge.md

ヘッダ出力されていること、確認。ちょっとコード引用してるあたり、style を何とかしたくはあります。
