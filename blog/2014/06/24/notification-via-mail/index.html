メイル通知盛り込み

ActionMailer Railsアプリからメールを送信 を見つつ盛り込み。devise からメイル送信云々なナニは既に盛り込み済みなのでとりあえず以下。

$ rails g mailer message hello

で、app/mailers/message.rb を以下なカンジにして

class Message < ActionMailer::Base
  default from: "noreply@example.com"

  def complete(url, to)
    @url = url
    mail to: to, subject: "mission complete"
  end
end
