DevOpsOkinawa #2

明日開催。わし的なメインテーマは Apache 掘削。バージョンは 2.4.7 との事なので入手しておくことに。

以下から、なのかどうか。

  • Download - The Apache HTTP Server Project

見てみるに 2.4.7 は latest available version らしい。とりあえず展開して gtags と思いきやパケジが導入されていない。

$ sudo apt-get install -fy global

導入してディレクトリ移動して

$ gtags -v

これで準備完了なのかどうか。エントリポイント探すのとかは土曜日で。

ちょっと追記

apr_sockaddr_info_get という手続きの定義が見えない。これはおそらくコンパイルしないと出てこないのだろうなと類推。configure に渡すスイッチを確認させて頂いておる次第です。
