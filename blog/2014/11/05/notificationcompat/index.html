自分メモ

実は最近の API でしか Notification な実装を作ったことがなくって API11 以前な SdkVersion なソレに盛り込むことになって、一体これはどうするのか、という事がありましたので備忘録な自分メモを投入。

最初は Build.VERSION あたりで切り分けなのかな微妙だな、って思っていたら SupportPackage に NotificationCompat というヤツがあるとのこと。

いやはや。

  • Android Tips #23 Android4.1 で追加された Notification のスタイルを使いこなす

背景透過

AndroidManifest で activity タグの属性として

android:theme="@android:style/Theme.Translucent.NotitleBar"

とかってやってあげると背景透過なテーマになる模様。これ、知りませなんだ。
