Docker-ce

自宅メイン端末で確認したところでは導入してるパケジは

$ dpkg --get-selections|grep docker
docker-engine                                   install

とのことだったのですが、新規導入な debian9 には見当らず。色々探してみるに Docker-CE なるソレを、という情報あり。CE は community edition とのこと。

以下、ドキュメントに沿って導入しました。

  • https://docs.docker.com/engine/installation/linux/docker-ce/debian/

ハロワは動きました。一般ユーザ権限で docker 起動する方法は以下。

  • https://docs.oracle.com/cd/E39368_01/e67079/section_rdz_hmw_2q.html

docker-compose

docker-compose は apt で導入しています。

自分メモ

docker-engine のバージョンは 1.11.2 とのことです。
