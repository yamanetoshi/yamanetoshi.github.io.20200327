Kotlin plugin 導入

とりあえず手元の端末に Android Studio は導入済みなので Kotlin Plugin の導入方法のみ、ということで。環境は OSX です。

  • Android Studio 起動
  • Configure -> Plugins 選択
  • 下に Install JetBrains Plugin というボタンがあるのでクリック
  • Kotlin という plugin が出てくるので選択して右に出てる Install ボタンをクリック
  • よしなに処理が終わると右に Restart なボタンが出てくるはずなのでクリックして Android Studio を再起動したら OK です

以下のリポジトリを clone して

  • https://github.com/ntaro/kotlin-android-hands-on

ビルドしてみましたが失敗。sync に失敗しているらしい。ええと、android-25 が、らしいので入れます。

  • Build Tools 26.0.0 が入ってないので入れます
  • SDK Manager 起動して色々導入
  • Build Tools 26.0.1 が導入されたので app/build.gradle の buildToolsVersion の記述を修正
  • ありゃ、AS が古いのか。な
  • http://qiita.com/k-yamada@github/items/b3544ab6ce7b708424cf
  • ずっと使ってないとハメてくれるな gradle めが
  • gradle な daemon 再起動が必要だった?
  • gradle のバージョンは 2.1.2 に設定 (app/build.gradle)
  • gradle-properties の distributionUrl のバージョンを 2.14.1 にしました (https://android.benigumo.com/20161005/agpbi-%E9%96%A2%E9%80%A3%E3%81%AE%E5%A4%9A%E9%87%8F%E3%81%AE%E3%82%A8%E3%83%A9%E3%83%BC%E3%82%92%E6%8E%92%E9%99%A4%E3%81%99%E3%82%8B/)
  • 失敗したが Install artifact and sync project でどうか
  • なんか Plugin Error でてますが Build はすすんでいる
  • なんとなく成功しているみたいなので再度 build 実行

Build はこれで動いたらしい。これでスタート地点に立てたのかどうか。つうかこれ、記録として酷い感満点だよ。
