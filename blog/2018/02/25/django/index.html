django いいな

ずいぶんご無沙汰ですが、以下なエントリ見るに勉強会開催してもいいんじゃないかと思い始めていたりしてる次第。

  • apache に django の web アプリをデプロイしてみよう
  • Django REST Framework を使って爆速で API を実装する

つうか

でびあんの勉強会で、ってのも色々な意味で面白いですねぇ。

あと

隙を見て雛形なコンテナイメージ作って dockerhub あたりに放り込んでおく方向で。

docker image 作成してみます

でびあんベースの docker てどうなのか。

$ docker pull debian

して

$ docker run -i -t debian /bin/bash

して以下。

# apt update
# apt install python3.5 python3-pip
# apt install apache2 libapache2-mod-wsgi-py3
# apt install sqlite3

で、さらに以下を実行して

# pip3 install django==2.0.2

イメージ作って dockerhub に push しています。

$ docker commit epic_colden debian-django
$ docker login
$ docker tag debian-django yamanetoshi/debian-django
$ docker push yamanetoshi/debian-django

これ、もう少し工夫が必要ですね。
