作業控え

なんというか、久々に弄ってみるに完全に忘れてるしメモも探せない。

とりあえず

python2.7 が動く環境ってことで

FROM python:2.7
MAINTAINER yamanetoshi <yamanetoshi@gmail.com>

みたいな Dockerfile こしらえて以下。

$ sudo docker build -t yamanetoshi/python:2.7 .

ちなみに Docker のバージョンは以下です。

$ sudo docker -v
Docker version 1.10.2 build c3959b1

イメージできてます

以下なカンジ。

$ sudo docker images |grep python
yamanetoshi/python                2.7                 422d625c4492        3 minutes ago       1.044 GB

で、こうなのかな。

$ sudo docker run -i -t yamanetoshi/python:2.7 /bin/bash

ちょっと必要なパケジを導入したりなど。

その後

以下で commit 作成するなど。

$ sudo docker commit -m 'install openjdk-7-jre' `sudo docker ps -l -q`

つうか最近、こうして控えておいても探せんのよねorz

追補

commit して image 作るのは良いのですがタグとか付けてなかった。以下にて。

$ sudo docker tag 16d32a643f6f yamanetoshi/hoge:latest
