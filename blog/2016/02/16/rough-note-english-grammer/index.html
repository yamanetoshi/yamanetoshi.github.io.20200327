English Grammer in Use 乱暴メモ (15)

Present perfect continuous (I have been doing)

現在完了進行形から。以下なやりとりが例示されています。

  • 雨降ってる?
  • や、でも地面は濡れてるな

ということで丁度今終わった、という状況で使うのかどうか。あるいは、ある期間続けてた、ということも表現できる模様。

Present perfect continuous and simple (I have been doing and I have done)

現在完了と現在完了進行の比較。今してたところ、てのといつだか分からんけどやりました、って差なのかどうか。あと

  • countinuous は how long (まだ何か、は続いてる)
  • simple は how much, how many, how many times (済んでるから量を聞くのか)

あるいは know/like/believe なんかは普通に使って進行形なナニになるとのこと。

How long have you (been) …?

present perfect は過去に始まって今もそれが続いてる何かについて話をする時に使うのか。

  • live と work はどちらでも使えます
  • always 付けるときは simple で
  • I haven’t done something since/for … という言い方

For and since

When …? and How long …? という題も付いてますね。

  • for と since を何かが起きてどの位経過したかを表現するために使える
  • for は省略できる (否定分では略せません)
  • for + all という使い方はしません
  • 否定文では for の代わりに in が使えます

when と how long の対比は過去形と現在完了の比較なのか。あと以下な言い方がある、とのこと。

  • It is {a long time / six month (etc.)} since something happened
  • It has been {a long time / six month (etc.)} since something happened
