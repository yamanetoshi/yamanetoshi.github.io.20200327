English Grammer in Use 乱暴メモ (12)

しまった。テキスト開いてみるに継続は Unit 140 でした。なんか見たことあると思ったら。ということで残りをぐわっとヤッツケて最初から頑張ります。

Phrasal verbs 4 on/off (1)

  • on/off は火を入れる、というソレ
  • イベントなどのための on/off
  • go on = happen
  • call something off = cancel it
  • put something off, put off doing something = delay it
  • clothes などのための on/off
  • put on / try on / take off
  • off = away from a person or place
  • be off (to a place)
  • go off とかの off (walk, run, drive, ride)
  • set off = start journey
  • take off = leave the ground (for planes)
  • see somebody off (見送り)

Phrasal verbs 5 on/off (2)

  • verb + on = continue doing something
  • drive on = continue driving (walk, play, etc.)
  • go on = continue
  • go on / carry on doing something = continue doing something
  • go on with / carry on with something
  • keep on doing something = do it continuously or repeatedly
  • get on = progress
  • get on (with somebody) = have a good relationship
  • get on with something = しなきゃいけない何かを続ける (通常割り込みを伴う)
  • doze off / drop off / nod off = fall asleep
  • finish something off = do the last part of something
  • go off = explode
  • go off = ring (alarm)

Phrasal verbs 6 up/doen

  • put something up / take something down
  • pick something up / put something down
  • stand up / sit down
  • turn something up / turn something down
  • break down = stop working
  • close down / shut down = stop doing business
  • let somebody down = disappoint them because you didn’t do what they hoped
  • turn somebody/something down = refuse an application, anoffer etc.
  • write something down

Phrasal verbs 7 up (1)

  • go up / come up / walk up (to …) = approach
  • catch up (with somebody), catch somebody up = move faster than somebody in front of you so that you reach them

  • keep up (with somebody) = continue at the same speed or level

  • set up a company (etc.) = start it

  • take up a hobby (etc.) = start doing it

  • fix up a meeting (etc.) = arrange it

  • grow up = become an adult

  • bring up a child = raise, look after a child

  • clean up / clear up / tidy up something = make it clean, tidy etc.

  • wash up = wash the plates, dishes etc. after meal

  • give up

  • take up spece or time = use space or time

  • turn up / show up = arrive, appear

  • use something up = use all of it so that nothing is left

Phrasal verbs 8 up (2)

  • bring up a topic etc. = introduce it in a conversation
  • come up = be introduced in a conversation
  • come up with an idea = produce an idea

  • make something up = invent something that is not true

  • cheer up = be happier

  • cheer somebody up = make somebody feel happier

  • clear up = become bright (for weather)

  • blow up = explode

  • blow something up = destroy it with a bomb etc.

  • tear something up = tear it into pieces

  • break up / split up = separate

  • do up a coat, a shoelace, buttons etc. = fastn tie etc.

  • do up a building, a room etc. = repair and improve it

  • hold up a person, a plan etc. = delay

Phrasal verbs 9 away/back

  • get away = escape

  • keep away (from …) = don’t go near

  • call (somebody) back = return a phone call

  • get back to somebody = reply to them by phone etc.

とりあえず

最初から見直しつつメモを取る方向にて。
