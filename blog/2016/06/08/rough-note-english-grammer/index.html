English Grammer in Use 乱暴メモ (21)

when I do / When I’ve done

When and if についても記述あり。

  • 未来形と一緒に使う When 句の中では未来形は使わない
  • つうか when の中では未来形使うな、なのかどうか
  • while / before / after / as soon as / until or til も同様とのこと
  • 現在完了も同様に使える
  • 現在完了使う場合、片方は他方の前に完了されていなければなりません
  • 同時に起きる場合は現在完了使えない
  • if は現在形を未来のために使える
  • when は必ず起こる事のために使います
  • if はおそらく起こるであろう事のために使います

ここはなかなか参考になります。

Can, could and (be) able to

ここも使い分けなナニなのかどうか。

  • can が意味する所について、あるいは通常は can (be able to ではなくて) を使うとのこと
  • could は can の過去形
  • 特に see, hear, smell, taste, feel, remember, understand と使う
  • 何かができてた、という意味でも使う
  • 特定の状況下で誰かが何かをできてた、という事が言いたい場合、could ではなくて was/were able to などを使う模様
