Will be doing and will have done

  • future continuous は何かをしている最中だろう、という表現
  • 未来における完全なアクションの表現
  • future perfect は未来のある時点で完了している何か、を表現

When I do / When I’ve done

  • when 節? では will は使わない
  • while/before/after/as soon as/until/til でも同様
  • 現在完了は when/after/until/as soon as の後で使える
  • if の後は未来を表現するために現在形を使う
  • if では起き得る事象のために使う
  • when では必ず発生する事象のために使う

Can, could and (be) able to

  • could は以下との組合せでよく使われる see/hear/smell/taste/feel/remember/understand
  • general ability, permission to do domething を誰かが持つ時に could を使う
  • 特定の場面で誰かが何かをした、ということを表現するときに was/were able to とか managed to を使う
  • couldn’t はどんな場面でも使える
