apery をでびあんで動かしてみる

何してるんだろ (ぇ

とりあえず

  • mono-complete 導入
  • gpsshogi 導入
  • shogidokoro.zip を download して解凍
  • apery を clone して make してみる

というあたりまで云々してみます。

mono-complete 導入したらもの凄い数のパケジが。そして gpsshogi というパケジは無いって言われましたね。sid にはあるらしい。これ、apery を使うのであれば不要かな。

む、なんか Problem: って出力が出てますね。そして Shogidokoro.zip を取得して解凍して

$ mono Shogidokoro.exe

したら落ちました。Ubuntu でやんないと、なのかどうかorz

Ubuntu で

リトライしてみます。apt-get update して以下。

$ sudo apt-get install gpsshogi mono-complete

よくよく見るにうぶんつも quantal ってことでとりあえず trusty にして dist-upgrade してみます。まぢで何してるんだろorz

ふむ

I2C なデバイスって並列に接続できるのか。他にソレ系の計測デバイスって無いかな。まだブレッドボードにはスキ間が十分にあります。

i2c 計測、でググッたら

  • 電圧、電流、電力計測
  • 温度、湿度
  • 液晶
  • 加速度、ジャイロ、コンパス

探してたら

pachube というキーワードが出てきた。これこれ。ということでこちらに push するナニをでっち上げることに。ログは別エントリにて。

その後

14.04 に upgrade して gpsshogi および mono-complete を導入着手。

$ mono shogidokoro.exe

が動作することは確認できてます。現在、apery.exe を取得すべく github なリポジトリに接続しているところです。

続きはまた別途ということで。
