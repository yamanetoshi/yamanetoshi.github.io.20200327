Kotlin

ちょっと IntelliJ 落として起動。

チュートリアル、やってみようかな。

  • Tutorials

とりあえず

Getting Started with IntelliJ IDEA から着手してみます。

Kotlin plugin を、なのかな。右下の Configure -> Plugins を選択。Install JetBrains Plugin を選択すると Browse JetBrains Plugins というソレが開くので Kotlin および Kotlin Extensions For Android というソレを導入すれば良いのかな。

画面閉じてくと Restart しろ、と言われるので仰せのままにすすめます。

再起動したら New Project を作れとのこと。JDK 指定して JVM は Kotlin (Java) を選択、とありますね。

開いてみるに Java から Kotlin (Java) が選択できます。ちなみに SDK は None というソレなんですが大丈夫なのかなぁ。あ、New から JDK 選択できました。選択して next すると Create project from template チェック入れたら Command Line App が選択されている状態になりました。

次に Project name などを、ということで HelloWorld という名前に。パケジはそのまんまで Finish なのかな。

あら、Main.java が開いたぞorz

間違えた

Java から Kotlin(Java) でわなくて左側に Kotlin という項目がありました。その中の Kotlin - JVM を選択して、どうなるか。

チュートリアルによれば src フォルダに適当な名前のファイルを作れ、とあるな。あ、app って名前にするのか。で、以下なカンジにして

fun main(args: Array<String>) {
    println("Hello, World!")
}

実行しましょう、とあります。で、中身を見てみるに Main.java ができてるなorz

リトライ

しても同じことになったので、java のソースファイルとパッケージを削除して app.kt を新規作成して実行したら Hello, World! 出ました。

Android 関連含めてチュートリアル確認してみようと思います。
