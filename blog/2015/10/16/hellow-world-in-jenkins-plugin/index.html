Jenkins Plugin なハロワ

以下の材料拝見しつつヤッてみました。

  • Jenkinsプラグインを作ってみる(1) - セットアップとサンプルコード -

mvn は諸々の download に時間がかかるのがアレです。

とりあえず

コンソールにハロワな出力、を確認。ここから諸々確認しつつメモを控える方向。

ええと

以下によれば

  • Plugin tutorial

エントリポイントは PluginImpl クラス、とのこと。がしかし、そげなクラスは見当たらないです。と思ったら直下に_多くの場合プラグインクラスは単に拡張ポイントを登録してるだけ_とありますね。

うーん

ハロワじゃ埓が開かないので以下エントリにあるナニを見てみます。

  • ChatWork用のJenkinsプラグインを作ってみた

とりあえず git clone して、mvn package してみます。それから import、で良いのかどうか。てか maven てディスクの消費が色々アレです。

そして

mvn package しても eclipse に import できませんね。むむむ、と言いつつ以下を実行。

$ mvn -DdownloadSources=true -DdownloadJavadocs=true -DoutputDirectory=target/eclipse-classes eclipse:eclipse

ええと、なんか disk full 寸前なんスけど。。とは言え、これで import 可能になりました。ちょっとこれ、ぐだぐだと読みつつメモを控える方向にて。
