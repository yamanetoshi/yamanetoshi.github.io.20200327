git-svn で色々アレだった件

使わせて頂いてた端末が故障して巨大な画面の端末を使わせて頂くことになったのですが、開発環境が全く無い状況。とりあえず、ということで

  • Chrome 入れて
  • adt-bundle な Eclipse 入れて

あとはリモートなリポジトリからソースツリー取得すれば、と思ったのですがここからなかなかにハメてくれたので以下に控えを。

とりあえず homebrew を入れて git 導入したのですが git svn clone できず。以下なカンジのエラーでした。

Can't locate SVN/Core.pm

とりあえず Xcode を導入して Command Line Tools 導入でコマンド自体は動くようになったもののネットワーク不具合で取得ができなくて困っていたのですが、何かのタイミングでそれに気づいてなんとかなった次第です。

以下なカンジで BASIC 認証なアレを登録? して

$ svn info --username fugahoge URL

あとは clone するだけでした。しかも Android プロジェクトの中に Git で管理されてるプロジェクトを仕込んでたりするのですが、.git はスルーで clone されてきました。何故だ。

教訓

OSX でプログラムする場合、最低限 Xcode と Command Line Tools は入れとかないと駄目、ってことで。
