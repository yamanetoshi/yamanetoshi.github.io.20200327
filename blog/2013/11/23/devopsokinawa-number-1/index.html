折角なので

わしもごそごそしてみることに。とりあえず仮想リソースを一つアレして、knife-solo で云々、ってあたりを纏めてみたいと思います。

まずは端末側の用意から、なのか。ディレクトリ掘って云々します。

$ mkdir devopsokinawa#1

で bundle init

$ bundle init

以下を追加して

gem 'chef'
gem 'knife-solo', :github => 'matschaffer/knife-solo'
gem 'berkshelf'

以下を実行。

$ bundle install --path vendor/bundle

あら、以下なエラーが出た。

An error occurred while installing nokogiri (1.6.0), and Bundler cannot continue.
Make sure that `gem install nokogiri -v '1.6.0'` succeeds before bundling.

色々試行錯誤したのですが、もしや、ということで以下。

$ xcode-select --install

導入失敗する。どうも今日は日が悪い模様。

Can’t Install the software because it is not currently available from the Software Update server.

これや駄目だな。いったんアキラメ。つうかこれってほぼ終わりなカンジですね。
