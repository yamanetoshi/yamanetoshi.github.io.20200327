UART 入力とか出力とか (2)

どうも enter が ‘\r’ (0x0d) なのが微妙、という話になっていたり。

文字列を入力して改行契機で出力、な例がありまして

    do {
        while(*UART0_FR & (1 << 4));
            c = * UART0_DR ;
            echo[length] = c & 0xff ;
            length ++;
    } while(c != '\n' && i < 256);

ここで改行を ‘\n’ (0x0a) としてますが自分が使ってる環境だと screen でも minicom でも ‘\r’ (0x0d) になってました。
