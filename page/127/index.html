labs.online.net 使ってみた

labs.online.net の invitation を頂戴したので、使ってみます。

とりあえず ssh のキーを登録したら dedicated なサーバを使えるらしい。とりあえず作ってみました。

  • 4 core @1.3GHz な CPU で 2GB なメモリらしい
  • Global な IP もらえるみたい
  • image は Ubuntu 14.04 がデフォですがでびあんも選択できる模様
  • volume は 100GB なソレが用意されている
  • これは別途 mount なのかどうか
  • system volume は 20GB なのか (SSD らしい)

でびあん選択で kickoff します。つうか専用物理ホストが、とか一体どんな仕掛けを使ってるのか非常に気になります。
